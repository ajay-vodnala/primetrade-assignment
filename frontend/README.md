## overview:

This project was developed as part of the Frontend Developer Intern Assignment for PrimeTrade.ai

It demonstrates skills in building modern, scalable, and secure web applications using React.js (Frontend) and Node.js & Express.js (Backend) with JWT authentication and MongoDB integration.

## Features Implemented:

Frontend (React.js)

---

-> Built using React.js with Bootstrap and css for a clean and responsive design.

-> Client-side form validation for registration and login pages.

-> Protected Routes implemented using React Router DOM.

-> Dynamic Dashboard to manage CRUD operations of tasks(sample entity)

-> Search and Filter options for better UX.

-> Logout functionality integrated.

Backend (Node.js + Express)

---

Lightweight backend with:

-> User Authentication (JWT-based)

-> Profile Fetch/Update APIs.

-> CRUD APIs for managing Tasks.

-> Password hashing using bcryptjs.

-> JWT middleware for route protection.

-> Connected to MongoDB for data persistence.

## Dashboard Features

-> Display logged-in user tasks.

-> Perform Create, Read, Update, Delete operations on their tasks.

-> Responsive UI supporting both desktop and mobile.

## Technologies Used

---

Frontend: React.js, TailwindCSS, Axios, React Router DOM
Backend: Node.js, Express.js
Database: MongoDB (Mongoose ORM)
Authentication: JWT (JSON Web Tokens)
Security: bcrypt.js for password hashing
API Testing: Postman
